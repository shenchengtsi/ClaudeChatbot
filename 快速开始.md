## 🚀 接下来的步骤

### 第一步：创建飞书机器人应用

1. 访问 https://open.feishu.cn/app
2. 点击「创建企业自建应用」
3. 填写应用名称（如：Claude Code Bot）
4. 进入应用后，左侧菜单选择「添加应用能力」→ 选择「机器人」

### 第二步：配置权限

在左侧菜单「权限管理」中，搜索并开启以下权限：
- `im:message`
- `im:message:send_as_bot`
- `im:message.group_at_msg:readonly`（如果需要在群聊中使用）

### 第三步：配置事件订阅

1. 左侧菜单「事件与回调」→「事件配置」
2. 选择「使用长连接接收事件」（重要！这样不需要公网IP）
3. 添加事件：`im.message.receive_v1`

### 第四步：获取凭证

1. 左侧菜单「凭证与基础信息」
2. 复制 **App ID** 和 **App Secret**
3. 编辑配置文件：
   ```bash
   open -e ~/feishu-claude-bot/config.py
   ```
4. 填入你的 App ID 和 App Secret

### 第五步：发布应用

在飞书开放平台点击「版本管理与发布」→「创建版本」→「申请发布」

### 第六步：获取你的 Open ID（安全白名单）

运行：
```bash
cd ~/feishu-claude-bot
./get_openid.sh
```

然后在飞书中给机器人发送任意消息，终端会显示你的 Open ID。

将 Open ID 复制到 `config.py` 的 `ALLOWED_OPEN_IDS` 列表中：
```python
ALLOWED_OPEN_IDS = [
    "ou_你的OpenID",
]
```

### 第七步：启动 Bot

```bash
cd ~/feishu-claude-bot
./start.sh
```

看到「🚀 飞书 Claude Code Bot 启动」即为成功！

## 🔧 开机自启（可选）

如果你希望 Mac mini 重启后自动运行 Bot：

```bash
# 复制配置文件
cp ~/feishu-claude-bot/com.feishu.claudebot.plist ~/Library/LaunchAgents/

# 加载服务
launchctl load ~/Library/LaunchAgents/com.feishu.claudebot.plist

# 查看状态
launchctl list | grep claudebot
```

停止自启：
```bash
launchctl unload ~/Library/LaunchAgents/com.feishu.claudebot.plist
```

## ❓ 常见问题

**Q: 提示找不到 claude 命令**
A: 已配置为 `/Users/samsonchoi/.local/bin/claude`，如果还有问题，运行 `which claude` 检查路径

**Q: 消息发出去没反应**
A: 检查飞书开放平台是否选择了「长连接」事件订阅，以及权限是否审核通过

**Q: 如何停止 Bot**
A: 在终端按 `Ctrl+C`

**Q: 如何修改默认工作目录**
A: 编辑 `config.py` 中的 `WORK_DIR` 参数
